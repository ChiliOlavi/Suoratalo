Television and Media Device Instructions

Welcome to Hotelli Suoratalo! We know that sometimes figuring out the TV or media devices in a new place can be a bit confusing. To help you unwind after a long day in Imatra, here are some straightforward instructions for using the television and related media devices in your room.

TV and Media Device Guide

Most of our rooms are equipped with a Solainet NexBox N system, with access to VIASAT and CANAL DIGITAL channels. Please read the instructions below carefully—if you need further help, Anna at the reception is always happy to assist (even if sometimes it takes a moment to figure out the remotes).

1. Solainet NexBox N – VIASAT / CANAL DIGITAL Selection

  • Remote Control:
    At the top of your remote, you will find buttons labeled “1” and “2”.
    • Button 1: Selects VIASAT
    • Button 2: Selects CANAL DIGITAL
    Press the desired button to switch between these two channel providers.
  • Smart Card:
    If prompted, insert the card labeled “3 OREXAN” into the slot on the NexBox.

2. Luminaris TaskMaster 3000

Some rooms have an additional media device named Luminaris TaskMaster 3000 (yes, the name is a bit much for a small hotel).

  • Power Button: Only use the power button if there is no picture, even though you have selected the correct HDMI input (see below).
  • Reset Button: Use the reset button only if absolutely necessary. If you need to reset, please follow the previous steps again after resetting.

3. HDMI Input Selection

  • TV Input:
    Make sure your TV input is set to HDMI1 (usually via the “Source” or “Input” button on the TV remote).
  • Aurinkomedia TV:
    The television should display the Aurinkomedia TV welcome screen if everything is set up correctly.

4. Savotin Pub Side – Sirpisat / Kanaali Nova Receiver

  • If you are staying in a room close to the pub or using the shared lounge TV:
    • Ensure the Sirpisat / Kanaali Nova receiver is turned on.
    • Select a channel that is available. If the screen remains blank, check that the receiver is on and the correct channel is chosen.

5. Troubleshooting

  • If you have followed all the steps and still have no picture, try the power button on the Luminaris TaskMaster 3000 one more time.
  • As a last resort, use the reset button, then repeat the above steps.
  • If you are still having trouble, please contact the reception or see our Error Reporting and Troubleshooting page.
